@media queries

CSS usses the @media rule to detect information about the device being used to access the website. For example, it can be used to detect the size of the device, its capabilities, and its orientation. For this course, we will be using it to detect the device size, but keep in mine that it is capable of much more.

Desktop: Width less than 1024

Tablet: Width between 481 and 1023

Mobile: Width greater than 480
